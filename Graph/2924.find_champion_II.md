# 2924.寻找冠军 II

题目链接：[2924.寻找冠军 II](https://leetcode.cn/problems/find-champion-ii/)

## 题目大意

一场比赛中共有 n 支队伍，按从 0 到  n - 1 编号。每支队伍也是 有向无环图（DAG） 上的一个节点

给你一个整数 n 和一个下标从 0 开始、长度为 m 的二维整数数组 edges 表示这个有向无环图，其中 edges[i] = [ui, vi] 表示图中存在一条从 ui 队到 vi 队的有向边

从 a 队到 b 队的有向边意味着 a 队比 b 队 强 ，也就是 b 队比 a 队 弱 

在这场比赛中，如果不存在某支强于 a 队的队伍，则认为 a 队将会是 冠军 

如果这场比赛存在 唯一 一个冠军，则返回将会成为冠军的队伍。否则，返回 -1 

注意
- 环 是形如 a1, a2, ..., an, an+1 的一个序列，且满足：节点 a1 与节点 an+1 是同一个节点；节点 a1, a2, ..., an 互不相同；对于范围 [1, n] 中的每个 i ，均存在一条从节点 ai 到节点 ai+1 的有向边
- 有向无环图 是不存在任何环的有向图

限制：
- 1 <= n <= 100
- m == edges.length
- 0 <= m <= n * (n - 1) / 2
- edges[i].length == 2
- 0 <= edge[i][j] <= n - 1
- edges[i][0] != edges[i][1]
- The input is generated such that if team a is stronger than team b, team b is not stronger than team a.
- The input is generated such that if team a is stronger than team b and team b is stronger than team c, then team a is stronger than team c.

## 解题

题目要求找出比赛中唯一的冠军选手，也就是：
- 没有被任何其他选手击败
- 所有其他选手最终都输给了这个人（即图中只有一个入度为 `0` 的点）
- 如果没有冠军或有多个候选人，返回 `-1`

本质上，这是一个`有向图`的问题，可以将比赛结果看成一张有向图：
- 边 `u -> v` 表示 `u` 击败了 `v`
- 那么被打败的节点 `v` 入度（被击败的次数）+1
- 想要成为冠军，入度必须为 `0`（没有人打败你）
- 若存在多个入度为 `0` 的人，则冠军不唯一，返回 `-1`

✅ 暴力解：
- 一开始可能会想：遍历每个节点，统计它击败了谁，并看它是否被别人打败
- 这种方法逻辑复杂，尤其是判断“是否没有被任何人击败”比较麻烦，需要双重遍历

**写法 1**

思路：`使用 入度`
- 如果只统计每个节点被打败的次数（即入度）：
  - 谁被打败了？入度 > 0
  - 谁没被打败？入度 == 0
  - 如果有且只有一个入度为 `0` 的节点，它就是冠军
  - 否则就没有冠军（返回 -1）
- 这正好符合图论中的入度定义，因此采用`统计入度`的方法

```js
var findChampion = function(n, edges) {
    const inDegree = new Array(n).fill(0);
    
    for (const [u, v] of edges) {
        inDegree[v]++;
    }

    const champions = [];
    for (let i = 0; i < n; i++) {
        if (inDegree[i] === 0) champions.push(i);
    }

    return champions.length === 1 ? champions[0] : -1;
};
```
```python
class Solution:
    def findChampion(self, n: int, edges: List[List[int]]) -> int:
        # 初始化一个长度为 n 的数组 in_degree
        # in_degree[i] 表示第 i 个选手被打败的次数（入度）
        in_degree = [0] * n
        
        # 遍历所有比赛结果 (u, v)，表示 u 打败了 v
        # 那么 v 的入度加一：in_degree[v] += 1
        for u, v in edges:
            in_degree[v] += 1
        
        # 遍历所有选手编号 0 ~ n-1
        # 把所有 没有被打败过的选手（入度为 0） 放入 champions 列表中
        champions = [i for i in range(n) if in_degree[i] == 0]
        
        # 如果 champions 中恰好只有一个人，说明只有一个冠军，返回该编号
        # 否则（没有或有多个），返回 -1
        return champions[0] if len(champions) == 1 else -1
```

- 时间复杂度: `O(m + n)`，其中：`m` 是比赛数量，即 `edges` 的长度；`n` 是选手数量
  - 遍历 `edges` 统计入度是 `O(m)`
  - 遍历 `in_degree` 查找入度为 `0` 的人是 `O(n)`
- 空间复杂度: `O(n)`，用于记录每个选手的入度
