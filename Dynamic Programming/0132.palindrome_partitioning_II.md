# 132.分割回文串 II

题目链接：[132.分割回文串 II](https://leetcode.cn/problems/palindrome-partitioning-ii/)

## 题目大意

给你一个字符串 `s`，请将 `s` 分割成一些子串，使每个子串都是回文串

返回符合要求的 最少分割次数 

```js
Example 1:
Input: s = "aab"
Output: 1
Explanation: The palindrome partitioning ["aa","b"] could be produced using 1 cut.

Example 2:
Input: s = "a"
Output: 0

Example 3:
Input: s = "ab"
Output: 1
```

限制:
- 1 <= s.length <= 2000
- s consists of lowercase English letters only.
